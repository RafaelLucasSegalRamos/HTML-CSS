Principais formatações

Aprenderemos como deixar um texto em NEGRITO/DESTAQUE

Está frase tera uma palavra em negrito utilizado a tag <B>(e é considerada não semantica)

Está frase tera uma palavra em destaque utilizado a tag <Strong>(e é considerada semantica)

Deixaremos um texto em Italico/Enfasê

Está frase terá uma palavra em Italico utilizando o <i>(e é considerada não semantica)

Está frase terá uma palavra com Enfasê utilizando o <EM>(e é considerada semantica)

Texto marcado

Podemos cria um texto marcado com a tag <mark>

Mas podemos mudar sua cor de diferente maneiras.

Textos GRANDES e pequenos

Criaremos textos grandes com a tag <Big> e textos pequenos com a tag <Small>

Textos deletados

Podemos Marcar um texto como e excluido para ser lido mas não considerado utilizado a tag <Del>

Texto Inserido

Podemos marcar um texto como inserido para dar enfase e ainda demosntrar que foi adcionado depois utilizado a tag <ins>

Sublinhado

Tambem existe a tag que pode sublinar texto que é a tag <u> (A tag <u> está se tornando obsoleta, ou seja, não semantica)

Texto SOBREscrito

Serve para colocar letras menores sobre as normais já escritas como em uma conta exemplo: x20+3=y com a tag <sup>

Texto SUBescrito

Para colocar letras menores embaixo das normais já escritas como em uma formula quimica H20 com a tag <sub>
